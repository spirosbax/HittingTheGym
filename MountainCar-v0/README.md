## Overview
Implementation of the Q-Learning algorithm for the MountainCar-v0 classic control enviroment using OpenAi-Gym.

## Dependencies
* python 3.5 or later
* numpy
* OpenAi gym https://github.com/openai/gym

## Usage
Run the file with
```sh
python3 mountainCar-v0.py
```
