## Overview
Implementation of policy gradient neural network using Keras (Tensorflow backend) in python3.

## Dependencies  
* python 3.5 or later
* Numpy  
* OpenAi Gym
* Keras
* Tensorflow

## Usage
Run the file with
```sh
python3 pong.py
```
