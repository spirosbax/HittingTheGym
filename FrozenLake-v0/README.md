## Overview
Implementation of the Q-Learning algorithm for the FrozenLake-v0 enviroment in OpenAI-Gym.

## Dependencies  
* python 3.5 or later
* numpy  
* OpenAi gym https://github.com/openai/gym

## Usage
Run the file with
```sh
python3 cartpole-v0.py
```
